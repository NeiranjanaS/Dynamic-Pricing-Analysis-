# Final Report

## Project Title

Travel, Tourism & Hospitality – Customer Retention and Dynamic Pricing Analysis

---

## Objectives

- Analyze hotel booking data to identify cancellation patterns.
- Study pricing trends across different seasons.
- Build a baseline machine learning model to predict booking cancellations.
- Develop an interactive Power BI dashboard for business insights.

---

## Methodology

- Cleaned and prepared the dataset using Python (Pandas).
- Performed SQL analysis to calculate booking KPIs and trends.
- Conducted Exploratory Data Analysis (EDA) using Matplotlib and Seaborn.
- Built Logistic Regression and Decision Tree models for cancellation prediction.
- Created an interactive Power BI dashboard to visualize key business metrics.

---

## Results

- Longer lead times were associated with higher cancellation rates.
- Non-refundable deposits showed the lowest cancellation rates.
- City Hotels experienced more cancellations than Resort Hotels.
- Peak seasons had higher Average Daily Rates (ADR).
- The predictive model successfully identified high-risk bookings.

---

## Recommendations

- Use dynamic pricing during periods of high demand.
- Target high-risk customers with personalized retention campaigns.
- Promote non-refundable booking options to reduce cancellations.
- Monitor booking trends regularly to improve revenue management.

---

## Conclusion

This project identified the key factors affecting hotel booking cancellations and pricing. The insights support data-driven decision-making for revenue optimization, customer retention, and future predictive analytics.
