# Model Insights

## Models Used

- Logistic Regression
- Decision Tree Classifier

---

## Model Performance

Both models were trained to predict booking cancellations.

The evaluation metrics included:

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

The model with the higher Accuracy and ROC-AUC is recommended for deployment.

---

## Important Features

The most influential features include:

- Lead Time
- Deposit Type
- ADR
- Market Segment
- Total Nights
- Hotel Type

These variables have the strongest impact on predicting booking cancellations.

---

## Business Recommendations

- Identify customers with long lead times and send reminder emails.
- Encourage non-refundable bookings through promotional pricing.
- Apply dynamic pricing during peak seasons.
- Monitor high-risk market segments such as Online Travel Agents.
- Use the trained prediction model to estimate cancellation probability before arrival.

---

## Conclusion

The predictive models provide a strong baseline for identifying cancellation risk. These insights can help hotels improve customer retention, reduce revenue loss, and support future AI-based dynamic pricing systems.
